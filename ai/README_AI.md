# Linear Regression AI Project

## Project Overview

This project implements a complete linear regression workflow in both Python and R. It analyzes the relationship between employee years of experience and salary using a shared CSV dataset, computes standard regression diagnostics, and produces annotated visualization outputs.

## Assignment Objective

Fit and evaluate a simple linear regression model that predicts **Salary** from **YearsExperience**. Report slope, intercept, Pearson correlation (r), Mean Squared Error (MSE), and R-squared, then visualize the data with a fitted regression line.

## Component Descriptions

### Python Notebook (`linear_model_python.ipynb`)

Interactive analysis notebook that loads the dataset, fits a scikit-learn linear model, computes diagnostics with scipy and sklearn metrics, and creates a matplotlib scatter plot with regression line annotations.

### R Notebook (`linear_model_r.ipynb`)

Interactive analysis notebook that loads the dataset, fits an `lm()` model, computes diagnostics in base R, and creates a ggplot2 scatter plot with regression line annotations.

### Python Script (`linear_model.py`)

Command-line regression tool that accepts a CSV filename and column names, validates inputs, prints statistics, and saves `regression_plot_python.png`.

### R Script (`linear_model.R`)

Command-line regression tool that accepts a CSV filename and column names, validates inputs, prints statistics, and saves `regression_plot_r.png`.

## Environment Setup

1. Navigate to the project directory:

   ```bash
   cd ai
   ```

2. Run the setup script:

   ```bash
   bash setupenv.sh
   ```

3. Activate the environment:

   ```bash
   conda activate linear_regression_ai
   ```

## Command-Line Execution Examples

Python:

```bash
python linear_model.py regression_data.csv YearsExperience Salary
```

R:

```bash
Rscript linear_model.R regression_data.csv YearsExperience Salary
```

## Expected Outputs

Both scripts print:

- Slope
- Intercept
- Pearson correlation coefficient (r)
- Mean Squared Error (MSE)
- R-squared
- Regression equation (`y = mx + b`)

Both scripts also save annotated scatter plots:

- `regression_plot_python.png`
- `regression_plot_r.png`

## Plot File Description

Each PNG file contains:

- A scatter plot of salary versus years of experience
- A fitted linear regression line
- An annotation box with the regression equation and Pearson r value
- Axis labels and a descriptive title

## Sample Workflow

1. Set up the environment with `bash setupenv.sh`.
2. Activate the conda environment.
3. Run the Python script to generate statistics and `regression_plot_python.png`.
4. Run the R script to generate statistics and `regression_plot_r.png`.
5. Launch JupyterLab and open both notebooks for interactive exploration:

   ```bash
   jupyter lab
   ```

6. Compare Python and R outputs to confirm consistent model results.
